# Automatic-Left-Ventricle-Segmentation-in-Myocardial-Perfusion-SPECT-Images
Myocardial perfusion SPECT (MPS) is commonly used to evaluate heart blood flow and detect coronary artery disease. Manual segmentation of the left ventricle is slow and prone to errors. Deep learning can automate this process, making it faster, more consistent, and reproducible.
